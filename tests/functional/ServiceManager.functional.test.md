# 🧪 Phase 3.2 功能测试用例

**测试范围**：服务管理 Tab + ServiceManager  
**测试类型**：手动功能测试  
**执行时间**：约 15 分钟  

---

## 📋 测试前准备

### 环境要求
- ✅ Obsidian 已安装
- ✅ Model Runner 插件已部署
- ✅ Search Relay 项目存在（D:\work\search-relay）
- ✅ Model Runner server 目录存在

### 启动步骤
1. 完全重启 Obsidian
2. 打开插件设置页面
3. 切换到"🔧 服务管理" Tab
