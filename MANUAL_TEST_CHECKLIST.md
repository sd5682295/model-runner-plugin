# Model Runner 插件 - 手动测试检查清单

**目的**：在实际 Obsidian 环境中验证插件功能  
**执行时间**：预计 30-45 分钟  
**测试环境**：Obsidian Desktop (Windows/Mac/Linux)

---

## 准备工作

### 1. 安装插件

- [ ] 将 `model-runner-plugin` 文件夹复制到 `.obsidian/plugins/model-runner`
- [ ] 确认以下文件存在：
  - [ ] `main.js`
  - [ ] `manifest.json`
  - [ ] `styles.css`
  - [ ] `server/server.js`
  - [ ] `server/config.json`

### 2. 启用插件

- [ ] 打开 Obsidian
- [ ] 设置 → 社区插件
- [ ] 关闭安全模式
- [ ] 在已安装插件中找到 "Model Runner"
- [ ] 点击切换开关启用

### 3. 确认插件加载

- [ ] 左侧 Ribbon 栏出现 CPU 图标
- [ ] 底部状态栏出现 "🔴 未运行"
- [ ] 按 Ctrl/Cmd + P 打开命令面板，搜索 "Model Runner"，能看到3个命令
- [ ] 打开开发者工具 (Ctrl/Cmd + Shift + I)，控制台无错误

**✅ 如果以上全部完成，继续下面的测试**

---
